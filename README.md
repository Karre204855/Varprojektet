# Varprojektet
Varprojekt i Webbutveckling. Alexander Karanzas, Marcus Ray och Liam Ottosson.

Vecka 9: Borta från lektionen (sjuk) denna vecka. Läste lite hemifrån och skapade en "ProfileMe" till profilen och denna "README".

Vecka 10: Vi har arbetat på våran penpot för att kunna bygga upp en protoyp av våran hemsida. Genom att göra detta så blir det tydligare hur vi vill att vår hemsida ska se ut och därför även lättare att veta hur vi ska programera hemsidan.
